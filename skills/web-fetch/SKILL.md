# Skill: web-fetch

当用户给出 URL 或要求查看网页内容时，按以下流程执行：

## 步骤

1. 用 `http_get` 工具拉取目标 URL 的原始内容（最多一次重试）。
2. 如果是 HTML，**只关注 `<title>`、`<h1..h3>`、正文段落**；忽略导航、页脚、脚本。
3. 如果是 JSON/纯文本，直接读取。
4. 输出格式：

   ```
   **标题**: <title>
   **来源**: <url>

   **摘要**（≤ 5 句中文）:
   - …

   **关键信息**:
   - 字段 1: 值
   - 字段 2: 值
   ```

## 限制

- 单次拉取不超过 200KB 文本，超出请截断并说明 "已截断"。
- 不要伪造无法验证的数字或日期；不确定就标 "未提及"。
- 抓取失败（4xx/5xx/超时）直接告诉用户失败原因，不要继续编造。
